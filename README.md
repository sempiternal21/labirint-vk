Основное взаимодействие происходит с клетками лабиринта. То есть, если имеем лабиринт 3х3, то у нас будут клетки 1, 2, 3...8, 9.
Клетки идут слева направо, сверху вниз.
Запуск осуществляется посредством команды "php -S 127.0.0.1:8001"
